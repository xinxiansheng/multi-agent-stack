# Heartbeat Configuration

## Schedule
Heartbeat fires every 1 hour.

## Tasks Per Heartbeat
Batch up to 4 checks per cycle. Prioritize by urgency.

## Default Checks
1. **Unread messages** — Check email/messaging for anything urgent
2. **Calendar** — Upcoming events in next 2 hours
3. **Agent health** — Verify all agents are responsive
4. **Pending tasks** — Review and remind about open tasks
